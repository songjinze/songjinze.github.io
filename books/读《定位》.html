近日读广告营销类经典书《定位》，对内容做些许总结。

# 核心观点

我个人大概总结为，《定位》一书观点核心在于，寻找战略方向需要由外向内，从外在的成果入手，寻找用户心智中对于产品的认知定位，进而确立内部发力点。

定位的秘诀，我认为就是寻找用户内心的”空位“，从而树立新的”定位“，抢夺新概念和产生新的影响力。

# 信息爆炸的时代

当今时代的信息化程度非常高，人们接受的广告、信息可以说已经到达了人脑接收的限度。从理论上讲，当人脑接收的信息到达一定程度后，就会触发”自我保护机制“，即不再接收新事物。

一个具像化的体现是，以前常说人最多能记住7位数字，那么一个人对于同一个商品品类在以前可能只会愿意排在买前几名的牌子。而随着信息爆炸，商品类目越来越多，人们对于同一个品类的选择可能只能缩短为前二三名，甚至毫不夸张地说只能选择第一名。

# “定位”方法

# 领导者跟随者

市场上任何一个商品品类总会有一个排名，可以根据排名粗略分为”领导者“和”跟随者“。”领导者“一般指排名第一，有领先地位。”跟随者“指排名二三名。领导者和跟随者的策略是不一样的。

“领导者”重在确立领先地位。

  • 保持竞争力，保持领先。
  • 用于承认失败，保持用户满意度。
  • 快速拦截一切竞争对手超过的可能性，即使投产比低，也不要给别人可乘之机。

“领导者”的优势在于”富者愈富，穷者愈穷“，保持领先地位的成本要远小于树立领先地位的成本。

”跟随者“的重点在于”见缝插针“。

  • 不仅给自己定位，也要给竞争对手定位，寻找竞争对手的弱点，进而树立自己的优势。
  • 避免跟风。
  • 寻找空位。例如高价低价、尺寸、性别等。

# ”定位“陷阱

  • 不要满足所有人的需求。
  • 不要盲目跟风。
  • 警惕无名陷阱。一个好的名字至关重要。
  • 警惕品牌延伸。一个好的定位会随着品牌延伸而失去重心，从而降低用户心中原有定位的价值。品牌延伸可以追求短期利益和迅速扩张，但代价一定是模糊不清的定位和降低品牌影响力。

# 个人人生定位

  • 定义好自己。
  • 起好名字。
  • 敢于犯错。或者说敢于试错。

”找匹好马“

  1. 公司。
  2. 老板。
  3. 朋友。
  4. 想法。
  5. 信心。
  6. 自己。

成功六部曲：

  1. 你拥有什么定位。
  2. 你想拥有什么定位。
  3. 你必须超越谁。
  4. 你有足够的钱吗。
  5. 你能坚持到底吗。
  6. 你的传播体现了自己的价值吗。

# 定位的素养

定位发挥作用，需要有好的语言”润色“，这和定位本身在人心智中寻找空位这个概念如出一辙。现代人大部分时间介于”非理性“和”理性“之间，因此同一件事情的好的语言描述还是可以产生一些微妙的效果的。

定位需要持之以恒。追求短期利益而见风使舵，往往没有很好的持续的效果，要有远见，有耐心，相信时间。

要保持敏锐，适应变化。这也是辩证地看待持之以恒地坚持定位这个问题，如果一个定位从全局来看已经是错的了，那么是不可能有成效的。

要客观，不能相信主观意见。必要时多找人、多找局外人看待问题。

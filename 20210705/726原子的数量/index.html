726原子的数量

2021-07-05  算法  LeetCode 

题目描述

给定一个化学式formula（作为字符串），返回每种原子的数量。

原子总是以一个大写字母开始，接着跟随0个或任意个小写字母，表示原子的名字。

如果数量大于 1，原子后会跟着数字表示原子的数量。如果数量等于 1 则不会跟数字。例如，H2O 和 H2O2 是可行的，但 H1O2 这个表达是不可行的。

两个化学式连在一起是新的化学式。例如 H2O2He3Mg4 也是化学式。

一个括号中的化学式和数字（可选择性添加）也是化学式。例如 (H2O2) 和 (H2O2)3 是化学式。

给定一个化学式，输出所有原子的数量。格式为：第一个（按字典序）原子的名子，跟着它的数量（如果数量大于 1），然后是第二个原子的名字（按字典序），跟着它的数量（如果数量大于 1），以此类推。

示例 1:

输入:
formula = “H2O”
输出: “H2O”
解释:
原子的数量是 {‘H’: 2, ‘O’: 1}。
示例 2:

输入:
formula = “Mg(OH)2”
输出: “H2MgO2”
解释:
原子的数量是 {‘H’: 2, ‘Mg’: 1, ‘O’: 2}。
示例 3:

输入:
formula = “K4(ON(SO3)2)2”
输出: “K4N2O14S4”
解释:
原子的数量是 {‘K’: 4, ‘N’: 2, ‘O’: 14, ‘S’: 4}。

解题思路

一道不错的栈 + map的思想。这里需要保证字典序的输出，因此可以使用TreeMap。

源码

class Solution {

    // 当前括号内 所有元素和数量
    private Stack<TreeMap<String, Integer>> treeMaps = new Stack<>();

    public String countOfAtoms(String formula) {
        int N = formula.length();
        TreeMap<String, Integer> curTreeMap = new TreeMap<>();
        for(int i = 0; i < N; ){
            char c = formula.charAt(i);
            if(c == '('){
                treeMaps.push(curTreeMap);
                curTreeMap = new TreeMap<>();
                i++;
            }
            else if(c == ')'){
                // 先看括号后面有没有数字
                int num = 0;
                int j = i + 1;
                while(j < N){
                    char a = formula.charAt(j);
                    if(a >= '0' && a <= '9'){
                        num = num * 10 + a - '0';
                        j++;
                    }else{
                        break;
                    }
                }
                i = j;
                if(num == 0){
                    num = 1;
                }
                // 将当前treeMap的所有内容乘以数字，然后放到栈顶map中，弹栈
                TreeMap<String, Integer> stackTop = treeMaps.pop();
                for(Map.Entry<String, Integer> entry : curTreeMap.entrySet()){
                    Integer preValue = stackTop.get(entry.getKey());
                    if(preValue == null){
                        preValue = 0;
                    }
                    stackTop.put(entry.getKey(), entry.getValue() * num + preValue);
                }
                curTreeMap = stackTop;
            }else{
                // 大写字母
                StringBuilder stringBuilder = new StringBuilder();
                stringBuilder.append(c);
                int j = i + 1;
                for(; j < N; j++){
                    char a = formula.charAt(j);
                    if(a >= 'a' && a <= 'z'){
                        stringBuilder.append(a);
                    }else{
                        break;
                    }
                }
                int num = 0;
                for(; j < N; j++){
                    char a= formula.charAt(j);
                    if(a >= '0' && a <= '9'){
                        num = num * 10 + a - '0';
                    }else{
                        break;
                    }
                }
                if(num == 0){
                    num = 1;
                }
                String s = stringBuilder.toString();
                Integer preValue = curTreeMap.get(s);
                if(preValue == null){
                    preValue = 0;
                }
                curTreeMap.put(s, num + preValue);
                i = j;
            }
        }
        StringBuilder output = new StringBuilder();
        for(Map.Entry<String, Integer> entry : curTreeMap.entrySet()){
            // 输出
            output.append(entry.getKey());
            if(entry.getValue() != 1){
                output.append(entry.getValue());
            }
        }
        return output.toString();
    }
}
